## 📘 Chapter 1. 🚀 “C언어 + 그림 = 재미”  
**raylib와 Visual Studio로 시작하는 그래픽 프로그래밍 첫걸음**

---

### 🎯 이 장에서 배우는 것
- C언어가 뭐고 왜 배우는지
- raylib이 뭔지, 왜 쓰는지
- Visual Studio를 설치하고 raylib로 그림을 띄우는 초간단 실습

---

### 👶 C언어는 뭐예요?

C언어는 **컴퓨터에게 정확하게 “무엇을 해라”라고 명령하는 언어**예요.  
게임, 로봇, 시스템, 앱까지 다 C언어로 만들 수 있어요.  
우리는 그냥 숫자만 다루지 말고, **“그림으로 보면서” C언어를 배워볼 거예요!**

---

### 🎨 raylib이 뭐예요?

`raylib`은 “C언어로 그림 그리기 쉽게 도와주는 도구”예요.  
우리가 직접 “공 그리기”, “움직이기”, “게임 만들기”를 가능하게 해줘요.

---

### 💻 Visual Studio는 뭐예요?

Visual Studio는 C언어 코드를 **쉽게 쓰고 실행할 수 있는 프로그램**이에요.  
이 안에서 코드 쓰고, 버튼 누르면 그림이 “팡!” 나타나요.

---

## ⚡ 따라하기: Hello! 공 그리기 (5분 컷)

---

### ✅ 1단계: 프로젝트 만들기
1. **Visual Studio 실행**  
2. `새 프로젝트 > 빈 프로젝트` 선택  
3. 이름 정하고 만들기

---

### ✅ 2단계: 소스 코드 추가하기
1. 오른쪽 '솔루션 탐색기'에서  
   **소스 파일 > 마우스 오른쪽 > 추가 > 새 항목 > C 파일(main.c)**  
2. 아래 코드 복사해서 붙여넣기:

```c
#include "raylib.h"

int main() {
    InitWindow(320, 240, "Hi");
    while (!WindowShouldClose()) {
        BeginDrawing();
        DrawEllipse(320/2, 240/2, 80, 80, BLUE);
        EndDrawing();
    }
}
```

---

### ✅ 3단계: raylib 설치 (NuGet 사용)

**붉은 줄이 보이면:**  
- `도구 > NuGet 패키지 관리자 > NuGet 패키지 관리`  
- `찾아보기` 탭 > "raylib" 검색  
- 설치 클릭!

---

### ✅ 4단계: 콘솔 창 없애기 (선택 사항)

**윈도우 창만 뜨게 하려면:**  
1. **프로젝트 > 속성 > 링커 > 시스템 > 하위 시스템 → Windows (/SUBSYSTEM:WINDOWS)**  
2. **링커 > 고급 > 진입점 → mainCRTStartup** 입력  

> 이렇게 하면 검은 콘솔 창 없이 깔끔하게 그림만 떠요!
